### Project Goals

To create a fast website that allows me to show off my projects and blog about my studies to solidify my knowledge.

### Libraries & Frameworks

* Gatsbyjs - A static site generator for React.
* React - A JavaScript library for building user interfaces.
* GraphQL - A data query language.
* Markdown - A plain text formatting syntax that can be easily converted to HTML.
* styled-components - Great CSS-in-JS library for styling components.
